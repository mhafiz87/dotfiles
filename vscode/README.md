# VSCODE

## Command Line Interface

```bash
code --list-extensions
```

```bash
# install pylance
code --install-extension ms-python.vscode-pylance
```

```bash
# uninstall pylance
code --uninstall-extension ms-python.vscode-pylance
```

```bash
code --update-extensions
```
